# Contributing to GitPulse

We welcome contributions! Please follow these steps to help improve GitPulse:

1. **Fork the repo** and create your branch from `main`.
2. **Setup your environment** using the instructions in `README.md`.
3. **Write clean, semantic Tailwind code**.
4. **Ensure PRs include a description** of what's changing.
5. **Issues**: If you find a bug (especially Prisma engine issues on Windows), please report it with full logs.

Happy shipping! 🚢
